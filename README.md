# Unicat_Online-Learning

## Feature by role

- Common Role
     - Login and register account
     - Update user information
     - Change password
- Student Role
     - Register course
     - View course details
     - View list of courses
- Admin Role
     - Manage and view chart courses
     - Manage and view chart users
     - View analytic
     - Block particular user, or student
